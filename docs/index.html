📌 软件介绍

Tenveo Camera Controller（腾为相机助手） 是一款 面向 Windows 平台的专业摄像头管理工具， 适用于 Tenveo 及标准 UVC 摄像头。

支持视频预览、PTZ 云台控制、图像参数调节、AI 跟踪、 RTMP 推流以及 OTA 固件升级。

✨ 核心功能

  • UVC 视频预览（H.265 / H.264 / MJPG / YUY2）
  • PTZ 云台控制、变倍、预置位
  • 图像参数与曝光控制
  • AI 单人 / 多人智能跟踪
  • RTMP 视频推流
  • 设备信息管理与 OTA 固件升级
  • 一键隐私保护（相机休眠）

💻 支持平台

✅ Windows 10 / Windows 11

✅ 标准 UVC Webcam & PTZ Camera
