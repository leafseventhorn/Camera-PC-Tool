软件介绍

Tenveo Camera Controller（腾为相机助手）是一款专业的 Windows 摄像头管理工具， 支持 Tenveo 及标准 UVC 摄像头。

软件集成视频预览、PTZ 控制、图像调节、AI 跟踪、RTMP 推流和 OTA 固件升级。

🎥 UVC 视频预览

支持 H.265 / H.264 / MJPG / YUY2

🎛 PTZ 云台控制

方向、变倍、预置位、聚焦

🎨 图像参数

亮度、对比度、饱和度、锐度

🤖 AI 跟踪

单人 / 多人智能跟踪

📡 RTMP 推流

推流到第三方直播平台

🔄 OTA 升级

在线固件升级，持续增强
