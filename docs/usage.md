# tenveo Camera controller 使用说明

欢迎使用 tenveo Camera controller！此文档将帮助你快速了解如何使用软件。

## 📌 1. 安装软件

1. 打开 GitHub Releases  
2. 下载最新版本 `TenveoCamController_Setup_vX.X.X.exe`  
3. 双击运行并完成安装向导
